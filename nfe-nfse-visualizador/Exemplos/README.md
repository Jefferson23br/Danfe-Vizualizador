# Exemplos de XML

Esta pasta **nao contem arquivos de exemplo versionados** por motivos de privacidade
e conformidade com a LGPD.

Para testar localmente:

1. Utilize XMLs oficiais autorizados pela SEFAZ (NF-e) ou pela prefeitura (NFS-e).
2. Prefira documentos de homologacao ou dados anonimizados.
3. Nunca publique notas fiscais reais em repositorios publicos.

Os arquivos `.xml` e `.pdf` desta pasta estao ignorados pelo `.gitignore`.
